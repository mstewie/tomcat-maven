# tomcat-maven
## Introduction
Simple project which uses a maven plugin to deploy and run a war file under Tomcat 7.

## Usage
### Run Tomcat
```
mvn tomcat7:run
```

### Stop Tomcat
```
mvn tomcat7:stop
```
## Default URL
```
http://localhost:8085/dsp-ad-server/index.jsp
```
